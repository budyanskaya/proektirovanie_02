# Задание 2. Используя «1С:Система проектирования прикладных решений» для выбранного предприятия, выполните описание Функций системы.
### Функции системы сгруппируйте по следующим категориям:
* Базовые процессы
* Ограничивающие процессы
* Обеспечивающие процессы
* Процессы производства
* Процессы управления развитием
В группе Базовых процессов и Ограничивающих процессов зафиксируйте как минимум по одному процессу (в качестве группирующей функции).
Для выбранных двух группирующих функций распишите Функции системы — должно быть представлено минимум по 5 функций для каждой группирующей функции.
### Требования к описанию Функций:
Для каждой функции необходимо указать:
* Наименование и краткое описание функции
* Результат функции
* Входящие данные и Потребителей (где применимо)
* Продемонстрировать варианты видов связей для входящих данных: «Вход» и «Управление»,
* Завести Роли и указать их в качестве Исполнителей функции.
### Требования к представлению:
Продемонстрируйте Схемы полученных функций, начиная с Группирующей функции, и далее переходя по всем детализированным Функциям системы.
